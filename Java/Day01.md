## 注释

```java
public class HelloWorld{
    public static void main(String[] args){
        //单行注释：只能注释一行文字//
        //out put Hello，World！
        System.out.println("Hello,World!");
        //多行注释：可以注释一段文字 /* 注释 */
        /*
        我是多行注释
        我是多行注释
        我是多行注释
        */
    }
}
```
